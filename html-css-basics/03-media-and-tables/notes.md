# Day 3 - HTML Media & Tables

## 📌 Media Elements

### 1. Image
- `<img>` is used to display images
- Important attributes:
  - src → image path
  - alt → accessibility text

### 2. Audio
- `<audio>` is used to play sound
- `controls` attribute adds play/pause UI

### 3. Video
- `<video>` is used to play videos
- Supports width, height, controls

---

## 📌 Tables

### Structure:
- `<table>` → main container
- `<thead>` → header section
- `<tbody>` → data section
- `<tr>` → table row
- `<th>` → header cell
- `<td>` → data cell

---

## 💡 Key Learnings

- Always use `alt` for images (accessibility)
- Use `<thead>` and `<tbody>` for clean structure
- Media elements need proper file paths
