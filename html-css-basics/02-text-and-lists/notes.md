# Day 2 Notes

## What I Practiced
- Headings (h1 to h3)
- Paragraphs
- Text formatting (strong, em, b, i)
- Ordered and unordered lists

## Key Learnings
- <strong> and <em> are semantic (better for accessibility)
- <b> and <i> are only visual
- Ordered lists are used for steps
- Unordered lists are used for grouping items
